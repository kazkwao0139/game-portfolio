1. 설계 철학

"게임은 퍼즐이 아니라, 이익과 리스크의 기회비용이어야 한다"

TCG는 30년간 같은 문제를 안고 있었다. 마나 스크류, 드로우 운빨, 원턴킬, 묘지 루프. 모든 TCG가 이 문제를 "땜빵"으로 해결해왔다.

Tear The Spellbook은 이 문제들을 땜빵이 아닌 시스템 레벨에서 해결한다.

핵심 원칙

  • 랜덤 최소화 - 운이 아닌 스킬로 승부
  • 정답 없음 - 매번 다른 선택, 매번 다른 기회비용
  • 구조적 해결 - 땜빵이 아닌 시스템 설계

3. 핵심 시스템

3.1 게임 구조

덱 → 패 → 필드 → 제외 (묘지 없음)

한 번 사용한 카드는 영구히 제외된다. 회수, 재활용 불가능. 마도서에서 페이지를 찢으면 다시는 돌아오지 않는다.

영역 설명 기존 TCG
덱 마도서 (드로우 소스) 동일
패 펼쳐진 페이지 동일
필드 시전된 마법 동일
제외 찢어진 페이지 (영구 소멸) 묘지 (회수 가능)

왜 묘지를 삭제했는가?

  • 묘지 회수 → 무한 루프의 원인
  • 무한 루프 → 원턴킬의 원인
  • 묘지 삭제 → 원턴킬 구조적 불가능

택티컬 소모전

🔥 핵심 게임플레이: 마도서가 점점 얇아진다

한 번 찢은 페이지는 돌아오지 않는다. 게임이 진행될수록 양측의 마도서는 얇아지고, 선택지는 줄어든다.

레이스 구조: 두 개의 시계

⏱️ 승리 조건

조건 결과
상대 라이프 0 승리

레이스의 본질:
"상대 라이프가 먼저 끝나냐, 내 덱이 먼저 마르냐"

카드를 쓸 때마다 = 총알을 쏘는 것
드로우를 할 때마다 = 탄창이 줄어드는 것
모든 행동이 양날의 검

라이프만 관리 라이프 + 덱 동시 관리
덱 아웃 = 거의 없음 덱 아웃 = 현실적 승리 조건
드로우 = 이득 드로우 = 이득 + 리스크

패스 시스템의 진짜 리스크

패스의 대가는 "템포 손해"만이 아니다.

선택 얻는 것 잃는 것
행동 템포, 덱 보존 패 적음
패스 패 +2장 템포 + 덱 2장 소모
패스 = 시간을 판다
+2장 드로우 = 죽음에 2장 가까워짐
상대는 그 턴에 나를 때린다

상황별 패스 판단

상황 패스? 이유
덱 여유 있음 고려 가능 자원 축적 여유
덱 얼마 안 남음 절대 안 됨 덱 아웃 위험
상대 덱이 더 적음 가능 버티기 전략
내 덱이 더 적음 안 됨 빨리 끝내야 함

삼각 상성: 덱 타입

🔺 어그로 vs 컨트롤 vs 밀덱

덱 타입 전략 승리 조건
어그로 빠르게 라이프 깎기 상대 라이프 0
컨트롤 버티면서 역전 상대 라이프 0 (후반)
밀덱 상대 덱 소모 유도 상대 덱 아웃

상성 관계:

  • 어그로 > 밀덱: 덱 마르기 전에 죽임
  • 밀덱 > 컨트롤: 버티는 동안 덱 말림
  • 컨트롤 > 어그로: 초반 버티고 역전

→ 가위바위보 구조 = 메타 순환 = 게임 장수

기존 TCG TTS
"어차피 회수됨" "이거 쓰면 끝임"
러쉬 → 원턴킬 자원 관리
콤보 암기 상황 판단
카드 재활용 매 장이 무게감
체스 + TCG
체스처럼 매 수가 무게감 있고, TCG처럼 다양한 선택지가 있다.
"지금 이 페이지를 찢을 것인가, 아낄 것인가?"

소모전의 심리적 효과

  • 긴장감: 자원이 줄어드는 것이 눈에 보인다
  • 무게감: 모든 선택에 책임이 따른다
  • 역전 가능성: 남은 카드 관리를 잘하면 역전 가능
  • 히치콕 서스펜스: "저 카드를 쓸까 말까" = 긴장의 연속

3.2 드로우 시스템

기본 수치

  • 덱: 30장
  • 시작 패: 3장
  • 매 턴 드로우: 랜덤 1장 + 선택 1장

왜 30장 / 3장인가?

설계 사고과정:

1. 소모전 컨셉 → 마도서가 얇아지는 게 체감되어야 함
2. 시작 패가 많으면 → 여유로움 → 소모전 느낌 약함
3. 시작 패 3장 → 빡빡함 → 매 장이 무게감
4. 매 턴 2장 드로우 → (30-3)/2 ≈ 13~14턴 게임
5. 짧고 빡빡한 소모전 = TTS 정체성
드로우 타입 설명 효과
랜덤 드로우 덱 위에서 1장 불확실성 유지 (TCG의 재미)
선택 드로우 덱에서 원하는 카드 1장 전략 보장 (운빨 제거)

서칭 카드가 필요 없는 이유

기존 TCG에서 서칭 카드는 "드로우 운빨 보완" 용도였다. 선택 드로우가 시스템에 내장되면, 서칭 카드의 존재 이유가 사라진다.

→ 덱의 모든 슬롯이 실전 카드로 채워진다.

패스 시스템: 턴 포기 → 추가 드로우

🔄 패스 시스템

턴 시작 시 드로우(2장) 후, 행동 대신 패스를 선택하면 랜덤 2장 추가 드로우

선택 드로우 행동 총 획득
행동 2장 (선택1+랜덤1) ✅ 가능 2장 + 행동
패스 2장 (선택1+랜덤1) + 2장 (랜덤) ❌ 불가 4장 (선택권 낮음)

왜 패스 시스템이 필요한가?

  • 패 말림 해결: 드로우 카드 없이도 자원 회복 가능
  • 전략적 선택지: 템포 vs 자원의 기회비용
  • 역전 가능성: 밀리고 있을 때 자원 축적 후 반격

트레이드오프 수학 모델

템포 vs 카드 어드밴티지 - TCG 이론의 핵심 개념

V(action) = 행동 1회의 기대 가치
V(card) = 카드 1장의 기대 가치

패스 트레이드오프:
행동 = V(action)
패스 = 2 × V(card)

밸런스 조건: V(action) ≈ 2 × V(card)

동적 균형: 상황에 따라 가치가 변함

게임 상황 V(action) V(card) 최적 선택
이기고 있음 높음 낮음 행동 (마무리)
지고 있음 낮음 높음 패스 (역전 카드 필요)
초반 중간 높음 상황에 따라 (셋업)
후반 높음 중간 행동 (결정타)

→ 상황에 따라 최적해가 바뀜 = 정답 없음 = 기회비용 설계 성공

3.3 자원 시스템

턴당 사용권: 5장

마나 카드 없음. "카드 사용권"이 자원이다.

카드 등급 사용권 소모 예시
일반 1 기본 유닛, 기본 스펠
강력 2 강화 유닛, 범위 스펠
궁극 3 피니셔, 필살기

마나 스크류가 불가능한 이유

  • 마나 카드 자체가 없음
  • 매 턴 자동으로 5 사용권
  • "마나가 안 온다" = 존재하지 않는 상황

기회비용 구조

궁극기 (3) + 일반 (1) + 일반 (1) = 5
강력 (2) + 강력 (2) + 일반 (1) = 5
일반 (1) × 5 = 5

어떤 조합이 정답인가? 정답 없음. 상황마다 다르다. 이것이 기회비용이다.

사용권 5개의 수학적 검증

덱 비용 분포 (일반적 구성)

비용 비율 30장 덱
1비용 40% 12장
2비용 35% 11장
3비용 25% 7장
평균 비용 1.85

턴별 패 흐름 분석

매 턴 사용 카드: 5 ÷ 1.85 ≈ 2.7장
매 턴 드로우: 2장
패 변화율: +2 - 2.7 = -0.7장/턴
턴 시작 패 사용 드로우 종료 패
1 3장 2.7장 0장 0.3장
2 0.3장 0.3장 2장 2장
3 2장 2장 2장 2장
... ... ... ... 빡빡함 유지

    ▼

사용권별 비교

사용권 평균 사용 패 변화율 결과
4 2.2장 -0.2장/턴 널널, 패 남음
5 2.7장 -0.7장/턴 빡빡, 패스 필요
6 3.2장 -1.2장/턴 너무 빡빡, 매턴 패스

패스 균형점

📊 검증 결과

  • 패스 주기: 약 3~4턴에 1번
  • 13턴 게임: 3~4번 패스 발생
  • 결론: 매턴 선택지 있음, 강제 아님

사용권 5개 = 스윗스팟

  • 4개 = 패 남아서 선택지 없음
  • 5개 = 빡빡해서 트레이드오프 발생
  • 6개 = 너무 빡빡해서 패스 강제

3.4 선후공 밸런스

항목 선공 후공
첫 턴 드로우 없음 없음
시작 패 3장 3장
선택 카드 1장 2장
랜덤 카드 2장 1장
장점 템포 덱 컨트롤

트레이드오프 구조

  • 선공: 먼저 행동, 하지만 원하는 카드 1장만
  • 후공: 늦게 행동, 하지만 원하는 카드 2장

둘 다 장단점이 있다 = 기회비용 = 정답 없음

확률론 검증

덱 구성과 키카드 비율이 정해지면, 선후공별 키카드 확보 확률을 계산할 수 있다. 템포 보정 계수를 적용하면 승률을 수치로 도출할 수 있으며, 이를 기반으로 50:50 밸런스 조정이 가능하다.

조합론 예시: 덱 30장 기준

변수 정의

변수 값 설명
D 30 덱 크기
K 3 또는 9 키카드 수 (덱 타입에 따라)
선공 선택 1장 확정 획득
선공 랜덤 2장 초기하분포
후공 선택 2장 확정 획득
후공 랜덤 1장 초기하분포

덱 타입 정의

덱 타입 키카드 수 특징
콤보덱 3장 (10%) 핵심 콤보 터지면 승리
굿스터프 9장 (30%) 다 좋음, 일반 대응

키카드 수에 따른 선후공 차이

키카드 수 선공 기대값 후공 기대값 차이
3장 (콤보) 1.14장 2.04장 0.90장
6장 1.28장 2.14장 0.86장
9장 (굿스터프) 1.55장 2.25장 0.70장
12장 1.79장 2.36장 0.57장
패턴: 키카드가 적을수록 선택권 가치가 높아지고, 후공이 유리해진다.
키카드가 많을수록 랜덤도 괜찮아지고, 선후공 차이가 줄어든다.

수학적 원리: 이것은 초기하분포의 분산 통제다. 선택 드로우가 분산을 0으로 만들고, 랜덤 드로우의 비율을 조절하여 전체 분산을 통제한다.

매치업별 분석

1. 콤보 vs 콤보

구분 선공 후공
키카드 기대값 1.14장 2.04장
장점 템포 콤보 확보

→ 후공이 키카드 0.9장 우위, 선공은 템포 우위. 밸런스 좋음.

2. 굿스터프 vs 굿스터프

구분 선공 후공
키카드 기대값 1.55장 2.25장

→ 선택권 가치 낮음 (어차피 다 좋음). 템포가 더 중요. 선공 살짝 유리.

3. 콤보 vs 굿스터프 (상성 매치)

상황 콤보 굿스터프
콤보 선공 빨리 터뜨려야 함 버티면서 대응
굿스터프 선공 버티다가 한방 콤보 전에 밀어야 함

→ 덱 특성 싸움. 긴장감 있는 매치업.

5. 비즈니스 모델

영역 역할 수익 구조
온라인 게임 (밸런스 통제) 구독료 / 시즌패스
오프라인 굿즈 (수집 욕구) 피지컬 상품

오프라인 굿즈 구조

  • 낱장 페이지: 개별 카드 판매
  • 마도서 바인더: 빈 표지 (가죽 장정 느낌)
  • 수집 목표: 페이지 모아서 마도서 완성
  • 프리미엄: 오버프레임 일러스트, 금박 이펙트
  • 한정판: 고대 마법사 에디션, 특별 표지

BM 철학

"카지노 사장이 될래, 넷플릭스 사장이 될래?"

  • 가챠 = 변동 수익 = 예측 불가 = 리스크
  • 구독 = 고정 수익 = 예측 가능 = 안정

게임 자체는 가챠 없이 구독 모델. 수집 욕구는 오프라인 굿즈로 충족.

PDA Map Design Guide

비주얼 & 공간 설계 원칙

CONFIDENTIAL

1. 개요

이 문서는 PDA 전장의 시각적 디자인과 공간 설계 원칙을 정리한 가이드입니다.

본편의 전장 구조(Chapter 5)가 '규칙'을 다룬다면, 이 문서는 '형태'와 '분위기'를 다룹니다.

참고: 이 문서는 전체적인 설계 방향을 제시합니다. 추후 테스트 과정에서 적합하지 않다고 판단되는 부분은 얼마든지 변경될 수 있습니다.

핵심 원칙: PDA는 디지털 세계를 배경으로 하며, 모든 비주얼 요소는 '플라스틱 기계장치'와 '정보 노이즈'의 느낌을 유지해야 합니다.

2. 크로스존 (정글) 설계

2.1 기본 구조

  • 중앙에 오브젝트 스폰 장소가 분지(Basin)처럼 파여있는 구조
  • 정글 몬스터 없음 — 기존 MOBA의 정글 파밍 개념 폐기
  • 앵커와 앵커를 잇는 길목 구조로 구성

2.2 시야 설계

분지 구조의 핵심은 비대칭 시야입니다:

위치 시야 특성
분지 안쪽 (오브젝트 스폰 지점) 바깥쪽 시야 제한 — 진입자가 불리
분지 바깥쪽 (길목) 안쪽 시야 확보 — 매복/견제 유리

설계 의도: 오브젝트를 먹으러 들어가는 팀이 시야적으로 불리하므로, 정보전(CCTV)의 가치가 극대화됩니다.

3. 노이즈 시스템 부쉬 대체

PDA는 전통적인 '부쉬(Bush)' 개념을 사용하지 않습니다. 대신 디지털 세계의 노이즈라는 새로운 시각 언어를 사용합니다.

3.1 노이즈의 시각적 특성

  • 글리치(Glitch) 효과가 공간에 떠있는 느낌
  • 정보가 깨지거나 왜곡되는 디지털 간섭 영역
  • 풀숲이 아닌 '데이터 노이즈'로 시야 차단을 표현

3.2 기능적 역할

노이즈는 기존 부쉬와 동일하게 시야 차단 및 은신 기능을 수행하지만, PDA 세계관에 맞는 비주얼로 표현됩니다.

4. 앵커 시스템

4.1 오픈 순서

  1. 초기: 1차 앵커만 해킹 가능
  2. 1차 점유 후: 해당 서킷*의 2차 앵커(좌/우) 열림
  3. 2차 앵커 1개 점유 시: 해당 방향의 3차 앵커 열림 (두 서킷 중 어디든 상관없음)
  4. 3차 점유 유지 시: 가디언 해킹 조건 충족

중요: 모든 앵커가 열린 후에는 어느 순서로 먹을지 각 팀의 자유 판단에 맡김.

4.2 앵커 대미지 설계

앵커 대미지 범위 특성
1차 앵커 ~1/초 (1/n) 기본 압박 수준
2차 앵커 증가 (1/n) 확대 밸류 가중
3차 앵커 최대 (1/n) 최대 진입 결계 (수비 인원 1명 이상 시 활성화)

설계 의도: 도트 대미지(압박 수준)로 설정하여 재탈환이 가능하도록 함. 즉사급 대미지가 아니므로 역해킹 시도가 의미 있음.

※ 모든 수치는 테스트 후 조정 예정

* 서킷: 기존 MOBA의 '레인(Lane)' 명칭 변경

4.3 가디언

가디언은 각 팀의 최종 거점이며, 상대 가디언을 해킹하면 승리한다.

항목 설명
위치 각 팀 베이스 중앙
해킹 조건 3차 앵커 점유 상태 유지 시 해킹 가능
대미지 도트 대미지 (3차 앵커보다 약하지만 충분히 강력)
활성화 조건 수비 인원 1명 이상 시 공격 활성화
범위 베이스 전역

변경 사항: 원본에서는 단일 대상 공격이었으나, PDA의 '범위 내 전체 공격 + 피해 분산' 원칙에 맞지 않아 도트 대미지 방식으로 변경되었습니다.

설계 의도: 가디언 대미지가 즉사급이 아니므로, 상대 베이스 진입 후에도 팀 협동을 통한 해킹 시도가 가능합니다. 단, 오래 버티기는 어려우므로 빠른 판단이 요구됩니다.

5. 오브젝트 & 콘솔 디자인

5.1 해킹 콘솔 형태

  • 원형 360° 콘솔 — 어느 방향에서든 해킹 가능
  • 위치 유불리 억제 — 특정 방향이 유리하지 않도록 설계
  • 베이스(콘솔) 위에 오브젝트가 올라가 있는 구조

5.2 오브젝트 형태 원칙

❌ 금지 ✓ 권장
무기 형태 (총, 검, 폭탄 등) 공간 장악을 상징하는 기계장치
파괴적 이미지 플라스틱 기계장치 느낌
유기체적 디자인 정보/회로/노드 이미지

5.3 토이 솔저 / 플라스틱 드래곤 연출

  • 스폰 상태: 베이스(콘솔)가 오브젝트를 홀로그램으로 위에 투영하는 연출
  • 해킹 완료: 홀로그램이 사라지고 메모리로 변환되어 해커에게 귀속

6. 서스펜스 설계 원칙

PDA 맵 디자인의 핵심 원칙:

"가치가 높은 곳일수록 노이즈와 장애물이 많다"

  • 고가치 지역(3차 앵커, 드래곤 스폰)에 진입할수록 시야가 제한됨
  • 자연스러운 긴장감 형성 — 보상이 클수록 리스크도 큼
  • CCTV(정보전)의 전략적 가치 극대화

7. 디멘션 브레이커 필드

디멘션 브레이커 스폰 시, 디버프 필드의 범위:

  • 분지 내부로 한정 — 브레이커 디버프가 맵 전체에 퍼지지 않음
  • 브레이커를 먹으러 들어가는 팀만 디버프의 영향을 받음
  • 분지 바깥에서 견제하는 팀은 디버프 없이 전투 가능

설계 의도: 브레이커 획득을 위한 진입 리스크를 명확히 하고, 바깥 팀의 견제 플레이를 유도함.
